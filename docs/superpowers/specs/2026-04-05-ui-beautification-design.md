# 自动记账 UI 美化设计文档

## 背景

App 目前功能完整，但 UI 朴素，缺乏视觉层次感。本次美化目标：温暖柔和风格，混合方案（MDCard + 自定义导航栏 + 主题色）。

## 配色与主题

- KivyMD 主题：primary_palette = "Orange"，theme_style = "Light"
- 背景色：米白 #FFF8F0
- 收入颜色：柔和绿 #66BB6A
- 支出颜色：柔和珊瑚红 #FF7043
- 导航栏：暖橙底色，当前页文字高亮白色，非当前页半透明白

## 卡片与布局

- 首页月度汇总：MDCard，圆角 12dp，轻微阴影，分两列显示收入/支出
- 首页账单列表：每条账单用 MDCard 包裹，圆角 8dp，左侧分类图标（彩色圆形背景），右侧金额
- 设置页分类列表：每个分类用 MDCard 包裹
- 统计页：图表区域和数据区域各用一个 MDCard 包裹
- 整体间距从 8dp 增加到 12dp

## 导航栏

- 高度从 56dp 增加到 64dp
- 每个按钮加图标（首页/统计/设置），图标在上文字在下
- 当前激活页面按钮加半透明白色圆角高亮背景
- 手动添加账单按钮移至首页右下角作为 FAB（悬浮按钮）

## 实现方案

- 方案：混合方案（MDCard 做卡片 + 保留自定义导航栏结构 + theme_cls 设置暖色主题）
- 不使用 MDBottomNavigation（之前在设备上崩溃过）
- 不使用 MDDialog（之前崩溃，已替换为 Kivy Popup）

## 涉及文件

- main.py：更新主题色、导航栏样式、添加 FAB
- src/ui/home_screen.py：汇总卡片、账单列表卡片、FAB
- src/ui/stats_screen.py：图表和数据区域卡片
- src/ui/settings_screen.py：分类列表卡片
