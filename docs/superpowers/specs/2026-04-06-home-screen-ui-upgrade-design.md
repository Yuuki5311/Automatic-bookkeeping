# 首页 UI 升级设计文档

日期：2026-04-06

## 目标

提升首页视觉层次，让摘要数据更突出，账单列表更易读。

---

## 摘要卡片

**样式：** 单张圆角卡片，橙色背景（`theme_cls.primary_color`），宽度占满屏幕，内边距 `16dp`，圆角 `12dp`，高度 `100dp`。

**布局：** 卡片内横向两列，左列显示"本月收入"标签 + 金额，右列显示"本月支出"标签 + 金额。所有文字白色。金额字号较大（`20sp`），标签字号较小（`13sp`）。

**实现：** 用 `MDCard` 替换当前的 `MDBoxLayout summary_box`，设置 `md_bg_color=primary_color`，内部嵌套横向 `MDBoxLayout`，左右各一个纵向 `MDBoxLayout`。

---

## 账单列表行

**样式：** 每条账单独立 `MDCard`，白底，圆角 `8dp`，行间距 `8dp`，内边距 `12dp`，高度 `72dp`。

**布局：**
- 左侧（`size_hint_x=0.65`）：上行显示商家名（正常字号），下行显示分类名（小字，灰色）
- 右侧（`size_hint_x=0.35`，右对齐）：上行显示金额（支出红色 / 收入绿色，字号 `16sp`），下行显示日期（小字，灰色）

**交互：** 点击整张卡片触发详情弹窗（与现有逻辑一致）。

**实现：** 将 `refresh()` 中的 `MDBoxLayout row` 替换为 `MDCard`，内部嵌套两列布局。
